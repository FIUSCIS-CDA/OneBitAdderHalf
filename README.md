#### MODULE NAME
       OneBitAdderHalf

#### DEPENDENCIES
       None

#### INPUTS
Name | Size(Bits)
-----|------------
A   |     1      
B   |     1      

#### OUTPUTS
Name | Size(Bits)
-----|------------
C  |     1      
S   |     1      

#### DESCRIPTION
       Computes  A  + B.  Stores two-bit result in C and S;  C is the
       first bit and S is the second.
